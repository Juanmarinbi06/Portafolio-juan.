Proyectos
1. ETL con Power Query
Proceso de extracción, transformación y carga de datos desde múltiples fuentes usando Power Query. Incluye limpieza, combinación de consultas y preparación de la data para su uso analítico.
Herramientas: Excel · Power Query

2. Dashboard de control semanal de producción
Dashboard en Excel para el seguimiento semanal de producción. Permite visualizar avances, identificar desviaciones y monitorear indicadores clave de forma rápida.
Herramientas: Excel · Power Query · Tablas dinámicas

3. Automatización con Power Query
Flujo automatizado de transformación de datos que elimina pasos manuales en la preparación de reportes. La data llega limpia y lista para analizar sin intervención cada vez.
Herramientas: Excel · Power Query

4. Cierre financiero por áreas — Power BI
Dashboard ejecutivo para el seguimiento de cierre de costos por área, con análisis Real vs. Presupuesto mensual y acumulado. Incluye matriz de cumplimiento por función y KPIs de producción y ventas.
Herramientas: Power BI · DAX · Power Query

5. Modelado de datos — Relación de tablas
Diagrama y estructura del modelo de datos usado en los proyectos de Power BI. Muestra cómo se relacionan las tablas y la lógica detrás del diseño.
Herramientas: Power BI · Modelado relacional
